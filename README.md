# 📖 Pokédex — Desafio Full Stack

> Aplicação Pokédex desenvolvida como desafio técnico, utilizando **Next.js** no front-end e **NestJS** no back-end, com deploy na Vercel.

🔗 **[Acesse o projeto ao vivo](https://pokedexdesafio-indol.vercel.app](https://pokedexdesafio-pnpz.vercel.app/))**

---

## 📄 Sobre o Projeto

Uma Pokédex web completa que consome dados de Pokémon, construída como um desafio de desenvolvimento full stack. O projeto explora a integração entre um front-end moderno em **Next.js** e uma API robusta com **NestJS**, ambos escritos em **TypeScript**.

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Camada | Uso |
|---|---|---|
| [Next.js](https://nextjs.org/) | Front-end | Framework React para renderização e roteamento |
| [NestJS](https://nestjs.com/) | Back-end | Framework Node.js para construção da API |
| [TypeScript](https://www.typescriptlang.org/) | Full Stack | Tipagem estática em todo o projeto |
| [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS) | Front-end | Estilização da interface |
| [Vercel](https://vercel.com/) | Deploy | Hospedagem e CI/CD do front-end |

## 🚀 Como Rodar Localmente

### Pré-requisitos

- [Node.js](https://nodejs.org/) v18 ou superior
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/hugoalmx/pokedexdesafio.git
   cd pokedexdesafio/Pokedex
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse no navegador: `http://localhost:3000`
